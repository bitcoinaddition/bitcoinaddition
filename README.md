### Coin Specs

• PoW Algorithm: Quark  
• Premine: 300,000 BTCA
• PoW Blocks: 1 - 500  
• PoS Blocks: Starting from 501  
• Block Time: 60 Seconds    
• Maturity: 20 Confirmations  
• Prefix: BTCA adresses start with the capital letter "B"  
• Ports: 4715 (p2p) / 45329 (rpc)

• Explorer http://explorer.btcadd.io

• Website [btcacoin.com](http://www.btcadd.io)
